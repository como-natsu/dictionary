# 辞書アプリ

Laravelを使用し作成した辞書アプリです。
キーワードとその説明を登録・検索が出来ます。

## 機能
- キーワードと説明の登録
- 登録内容の一覧表示(最新順)
- 登録した内容の更新・削除
- キーワードや説明の検索
- 入力値のバリデーション(必須・文字数制限)

## 使用技術
- PHP 7.4.9
- Laravel 8.83.29
- MySQL

## インストール方法

1. リポジトリをクローン
2. composer install を実行
3. `.env.example`をコピーして`.env`を作成
4. `php artisan key:generate`を実行
5. `.env`ファイルにデータベース情報などを設定
6. `php artisan migrate`を実行
